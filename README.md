# myReactJSChallenger
Creating a simple graphic using codemirror editor with JSON command example code
## How to run this example
1. First run 
```
git clone https://github.com/devfabiosilva/myTeslaSMPS.git
```
2. Type
```
npm install
```
to install all dependencies
3. Type
```
npm start
```
